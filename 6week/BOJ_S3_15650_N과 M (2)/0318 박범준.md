# BOJ_S3_15650_N과 M (2)

[문제링크](https://www.acmicpc.net/problem/15650)

### 분류
백트래킹


### 접근법

### Code
```python
n,m = list(map(int,input().split()))
s = []
def dfs(start):
    if len(s)==m:
        print(' '.join(map(str,s)))
        return
    
    for i in range(start,n+1):
        if i not in s:
            s.append(i)
            dfs(i+1)
            s.pop()
```
